============================================================
        数学复习笔记：从抛物线到斜坐标系
        （直角坐标 / 极坐标 / 向量 / 复数 / 斜坐标 的统一线索）
============================================================

说明：公式用纯文本写，x^2 表示 x 的平方，sqrt 表示根号，
* 表示乘，omega(ω) 是两轴夹角。每一节都配了一张 svg 图。


------------------------------------------------------------
一、直线与抛物线的两交点（图：line_parabola.svg）
------------------------------------------------------------
抛物线 y = a x^2 + b x + c，与一条直线交于横坐标为 x1、x2 的两点。
过这两个交点的直线为：
    y = [a(x1+x2) + b] * x + (c - a*x1*x2)
要点：
- 斜率 k = a(x1+x2) + b
- 截距 m = c - a*x1*x2
- 顶点在原点的特例 y = a x^2 时：y = a(x1+x2)x - a*x1*x2
- b 只影响斜率，不影响截距。


------------------------------------------------------------
二、韦达定理 / 根与系数关系（图：vieta.svg）
------------------------------------------------------------
二次方程 a x^2 + B x + C = 0 若有两根 x1、x2：
    两根之和 x1 + x2 = -B/a
    两根之积 x1 * x2 = C/a
来源：a(x - x1)(x - x2) 展开后与原式逐项对比。
应用：直线与抛物线相交，把 ax^2+bx+c = kx+m 整理成
    a x^2 + (b-k)x + (c-m) = 0
其两根就是交点横坐标，套韦达定理即得第一节的公式。
几何直觉：对称轴 x = -B/(2a) 恰在两根正中间。


------------------------------------------------------------
三、初中“快速武器”小结
------------------------------------------------------------
1. 韦达定理：涉及两交点/弦，不解根直接用和与积。
2. 对称轴 x = -b/(2a)，顶点纵坐标 (4ac-b^2)/(4a)。
3. 两交点定直线（见第一节）。
4. 弦长：|x1-x2| = sqrt(b^2-4ac)/|a|，斜线弦长再乘 sqrt(1+k^2)。
5. 平移：y = a(x-h)^2 + k，左加右减、上加下减。
注意：抛物线的“焦点、准线”是高中圆锥曲线内容，初中不考、解答里别用。


------------------------------------------------------------
四、参数法 / 设点法（图：param_point.svg, param_isosceles.svg, similar_k.svg）
------------------------------------------------------------
“参数方程”（曲线的 x=f(t),y=g(t)）是高中内容；
但“设参法/设点法/设而不求”的思想初中天天用：
- 比例：设 a=2k, b=3k（k 最后会约掉）。
- 应用题：工程设总量为 1，行程设速度或时间。
- 几何：设线段长为未知数列方程。
- 二次函数压轴：设动点 P(t, at^2+bt+c)。

设点法典型（铅垂高求面积最大）：
  抛物线 y=-x^2+2x+3，A(-1,0) B(3,0) C(0,3)。
  设 P(t, -t^2+2t+3)，直线 BC: y=-x+3。
  铅垂高 h = (-t^2+2t+3) - (-t+3) = -t^2+3t。
  S(PBC) = (1/2)*底*铅垂高 = (1/2)*3*(-t^2+3t)，t=3/2 时最大 = 27/8。

设点法（对称轴上设点，等腰分类讨论，图 param_isosceles.svg）：
  对称轴 x=1，设 Q(1,m)。BC = 3*sqrt2。
  QB=QC -> m=1
  QB=BC -> m = ±sqrt14
  QC=BC -> m = 3±sqrt17
  共 5 个 Q，注意排除三点共线 (1,2)。

设 k 法（相似面积比，图 similar_k.svg）：
  DE//BC，AD:DB=2:3 -> 设 AD=2k,DB=3k -> AD:AB=2:5。
  相似比 2:5 -> 面积比 4:25 -> 三角形 ADE : 梯形 = 4:21。


------------------------------------------------------------
五、极坐标的“影子”：方位角与解直角三角形
（图：polar_bearing.svg, bearing_reef.svg）
------------------------------------------------------------
极坐标 (r,θ) 是高中内容，但其灵魂“用距离+角度定位点”在初中
就是解直角三角形（方位角、仰角俯角、坡度）。核心转换：
    x = r*cos(θ),  y = r*sin(θ)

方位角应用题（船是否触礁，图 bearing_reef.svg）：
  船由西向东。A 处测 P 北偏东60°，行 AB=10 到 B 测 P 北偏东30°，礁区半径 8。
  转角：∠PAB = 90-60 = 30°；∠PBA = 180-(90-30) = 120°；
  内角和 -> ∠APB = 30° -> 等腰 BP = AB = 10。
  作垂线 PD = BP*sin60° = 5*sqrt3 ≈ 8.66 > 8 -> 不触礁。
套路三步：方位角转内角 -> 用等腰/相似求一条边 -> 作垂线解直角三角形。


------------------------------------------------------------
六、复数与极坐标（图：complex_polar.svg, complex_mult.svg）
------------------------------------------------------------
复数 z = a + b i 的极坐标形式：
    z = r(cosθ + i*sinθ) = r * e^(iθ)
    模 r = sqrt(a^2+b^2)，幅角 θ；且 a = r*cosθ, b = r*sinθ（同一套转换）。

乘法 = 模相乘、幅角相加（= 旋转 + 伸缩）：
    z1*z2 = r1*r2 * [cos(α+β) + i*sin(α+β)]
    例：2∠30° × sqrt2∠45° = 2sqrt2∠75°。乘以 i = 逆时针转 90°。

棣莫弗定理：[r(cosθ+i sinθ)]^n = r^n (cos nθ + i sin nθ)。
z^n = 1 的 n 个根均匀分布在单位圆上（单位根）。
欧拉公式：e^(iθ) = cosθ + i*sinθ（指数、三角、复数、极坐标焊在一起）。


------------------------------------------------------------
七、向量与极坐标（图：vector_polar.svg, polar_unit_vectors.svg）
------------------------------------------------------------
向量的极坐标形式 = 大小 + 方向，分解：
    vx = r*cosθ,  vy = r*sinθ（力的分解就是这个）。例：F=10N, θ=37° -> Fx=8, Fy=6。

关键对比：
- 复数擅长“乘法/旋转”（幅角相加）。
- 向量主要做“加法”，不能直接模相加，要按分量加（平行四边形法则）。

极坐标基底向量（圆周运动，图 polar_unit_vectors.svg）：
    e_r（径向，沿 OP 向外），e_θ（切向，逆时针）。
    位置沿 e_r；速度沿 e_θ（切线方向）；向心加速度沿 -e_r（指向圆心），大小 v^2/r。


------------------------------------------------------------
八、斜坐标系（图：oblique_basis.svg, oblique_grid.svg, oblique_vector_basis.svg）
------------------------------------------------------------
斜坐标系 = 用两个不垂直的基底 e1、e2 给平面定坐标。
理论基础 = 平面向量基本定理：任意向量唯一写成 v = x*e1 + y*e2，(x,y) 即斜坐标。
要点：
- 求斜坐标 = 向量分解，投影“平行于另一轴”，不是垂直投影。
- 向量本身不变，坐标随基底改变；直角坐标是“基底垂直且等长”的特例。
- 取图形的边为基底：平行四边形 -> 顶点 (0,0)(1,0)(0,1)(1,1)；
  三角形 -> A(0,0) B(1,0) C(0,1)。
- 长度公式带交叉项：|v|^2 = x^2 + y^2 + 2xy*cos(omega)（单位基底）。
  omega=90° 时交叉项消失，退回 x^2+y^2。


------------------------------------------------------------
九、斜坐标+向量的六大应用点：分“两大类”记
（图：oblique_area.svg = 仿射类, oblique_metric.svg = 度量类）
------------------------------------------------------------
分界线：去掉“垂直”只影响“量长度/求角度”，不影响“线性组合/比例”。

第一类  仿射性质（不需要 cos omega，斜坐标可照搬直角坐标公式）：
  (1) 直线方程：仍是线性式 a x + b y + c = 0，两点式/参数式照用。
  (2) 共线判断：向量 (a,b)、(c,d) 共线 <=> a d - b c = 0；
      三点共线 <=> (x2-x1)(y3-y1) - (x3-x1)(y2-y1) = 0。
  (3) 面积比：实际面积 = |行列式| * S0，S0 = |e1×e2|。
      求比值时 S0 抵消 -> 面积比 = 行列式之比，跟直角系算法一样，免 omega。
  (4) 平行四边形/三角形坐标化：取边为基底让顶点坐标最简。

第二类  度量性质（必须用“点积矩阵 G”，带 cos omega）：
  (6) 点积矩阵  G = [ e1·e1  e1·e2 ]  单位基底时 = [ 1     cosω ]
                    [ e2·e1  e2·e2 ]              [ cosω  1   ]
      u·v = (a,b) G (c,d)^T = a c + b d + (a d + b c)*cos omega。
      垂直: u·v = 0；夹角: cosθ = (u·v)/(|u||v|)。
  (5) 距离公式: |PQ|^2 = Δx^2 + Δy^2 + 2 Δx Δy cos omega（单位基底）；
      一般: p^2 Δx^2 + q^2 Δy^2 + 2 p q cosω ΔxΔy（p=|e1|,q=|e2|）。

口诀：问“在不在线上 / 占多少比例 / 怎么分割” -> 放心用坐标硬算；
      问“多长 / 多少度 / 是否垂直” -> 立刻请出点积矩阵 G。

十、综合例题（图：oblique_example.svg）—— 两类同时用
------------------------------------------------------------
三角形 ABC，|AB|=3，|AC|=2，∠BAC=60°。E=AB 中点，F 在 AC 上 AF=(2/3)AC。

建系：取 e1=AB, e2=AC。则 A(0,0) B(1,0) C(0,1) E(1/2,0) F(0,2/3)。

(1) 面积比【仿射，免 cosω】：
    S(AEF):S(ABC) = (AE/AB)(AF/AC) = (1/2)(2/3) = 1/3。
    （用行列式：1/6 比 1/2 = 1/3，全程不用 60°。）

(2) 求 |EF|【度量，用点积矩阵】：
    e1·e1=9, e2·e2=4, e1·e2 = 3*2*cos60° = 3。
    EF = F - E = -(1/2)e1 + (2/3)e2。
    |EF|^2 = (1/2)^2*9 + (2/3)^2*4 + 2*(-1/2)*(2/3)*3
           = 9/4 + 16/9 - 2 = 73/36。
    |EF| = sqrt(73)/6 ≈ 1.42。
    （交叉项 e1·e2=3 绝不能漏。）


------------------------------------------------------------
一条主线总结
------------------------------------------------------------
直角坐标：正交等长基底 (i,j) —— 最基础。
极坐标：用“距离+角度”描述同一点 —— 换参数化。
向量：有大小有方向，可在任意基底下分解。
复数：绑定正交基底 {1,i}，额外获得“乘法=旋转”。
斜坐标系：最一般的线性坐标框架，去掉“垂直”，直角系是其特例。
=> 它们是“如何给平面上的点/向量建坐标”这一主题的五个侧面。

------------------------------------------------------------
十一、导数入门（图：secant_tangent.svg, extrema_cubic.svg）
------------------------------------------------------------
导数的定义（割线斜率的极限 = 切线斜率）：
    f'(x) = lim(h->0) [f(x+h) - f(x)] / h
以 y=x^2 在 P(1,1)：Q->P 时割线斜率 3.5->3->2.5->2.25->2 = f'(1)。
几何意义：f'(x) 就是曲线在 x 处切线的斜率。对 y=x^2 有 f'(x)=2x。

常用求导公式：
    (x^n)' = n*x^(n-1)，(常数)'=0，(sin x)'=cos x，(cos x)'=-sin x，和差逐项求导。

导数求单调性与极值（f(x)=x^3-3x）：
    f'(x) = 3x^2 - 3 = 3(x^2-1) = 3(x-1)(x+1)
    f'>0 递增，f'<0 递减，f'=0 处取极值。
    x<-1 增；-1<x<1 减；x>1 增 => 极大(-1,2)，极小(1,-2)。
求极值三步：1) 求 f'(x)；2) 解 f'(x)=0；3) 判两侧 f' 正负(增->减极大，减->增极小)。



------------------------------------------------------------
十二、导数的实战：优化应用题（图：box_net.svg, box_volume.svg）
------------------------------------------------------------
题: 边长12正方形铁皮, 四角剪去 x*x, 折成无盖盒子, 求体积最大。
  底面 (12-2x), 高 x, 体积 V(x)=x(12-2x)^2, 定义域 0<x<6。
  V(x)=4x^3-48x^2+144x
  V'(x)=12x^2-96x+144=12(x-2)(x-6)
  令 V'=0 -> x=2 或 x=6(边界舍)。
  0<x<2: V'>0 增；2<x<6: V'<0 减 -> x=2 为最大。
  V(2)=2*8^2=128。结论: x=2 时体积最大 128。

优化套路四步:
  1) 设变量, 写目标函数 + 定义域(带不等式限制)
  2) 求导, 令 f'=0 解候选点, 剔除定义域外
  3) 用导数符号判定(先增后减=极大/最大)
  4) 代回算最优值, 回答实际问题
适用: 用料最省 / 利润最大 / 容积最大 / 路程最短 等。
